# 👋 Hi there, I'm Youngha Choi

### 🚀 Junior Backend & DevOps Engineer
백엔드 개발과 API 성능 최적화, 클라우드 인프라 설계 및 운영 자동화에 관심이 많습니다. Spring Boot, Spring Data JPA를 활용한 백엔드 개발 경험을 바탕으로 AWS EKS, Kubernetes, Jenkins 등을 활용한 CI/CD 파이프라인 구축 및 운영을 탐구하고 있습니다.
